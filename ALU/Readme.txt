Este es el proyecto basado en una calculado en el cual solo imprime el infijo y el postfijo de lo que es un pila la cual puede recibir parametros ya sean operaciones matematicas
como por ejemplo (a+b)/(c-d).

En el cual declaramos la entrada de los datos la cual la almacenamos en  un pila y tenemos  3 pilas la primera pila trata de los datos iniciales que ingresaremos
la segunda seria una pila temporal en la cual almacenariamos los datos de la inicial para poder irlos a acomodando en la ultima pila que es la que seria la postFija
El codigo esta bien documentado se explica al final tambien la jerarquia que se le da a los operando para que el el resultado postfijo salga bien.
Tambien utilizando un poco de java scrip fue facil lograr imprimir el resultado de la operacion.
Este progama fue escrito en java utilizando la libreria scanner para poder leer, y la libreria Stack que es de las pilas las plataforma en la que se hizo fue en Windows.
